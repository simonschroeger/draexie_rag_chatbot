4 · RAG Query — Sequence Detail

sequenceDiagram participant U as 👤 User participant FE as Frontend (React) participant BE as FastAPI participant OL as Ollama · Gemma4 participant QD as Qdrant participant DB as SQLite U->>FE: Types question → Enter FE->>BE: POST /chat {question, conversation_id, file_ids?} BE->>OL: _expand_chain — generate 2 alt. phrasings OL-->>BE: ["alt query 1", "alt query 2"] loop 3 queries (original + 2 alternatives) BE->>QD: hybrid_search(k=6, BGE-M3 + BM25 + RRF) QD-->>BE: ranked docs with scores end BE->>BE: deduplicate by page_content BE->>BE: typed sort → text_docs[:6] + figure_docs[:2] BE->>BE: format_docs() → [1]…[N] numbered context BE->>DB: SQLChatMessageHistory.get_messages() DB-->>BE: full conversation history BE->>OL: stream(system_prompt + history + context + question) loop streaming tokens OL-->>BE: token chunk BE-->>FE: SSE data: {"token": "…"} FE-->>U: renders incrementally + blinking cursor end OL-->>BE: generation complete BE-->>FE: SSE data: {"sources": […], "chunks": […]} BE-->>FE: SSE data: {"suggestions": ["?","?","?"]} BE-->>FE: SSE data: {"done": true} BE->>DB: save AI message to message_store BE->>DB: save sources/chunks to message_sources FE-->>U: citation badges [N] rendered FE-->>U: Knowledge Panel available FE-->>U: suggestion chips shown alt First message in conversation FE->>BE: POST /conversations/{id}/title BE->>OL: "Erstelle Titel (max 6 Wörter): {first_message}" OL-->>BE: German title BE->>DB: INSERT conversation_titles BE-->>FE: {"title": "…"} FE-->>U: Sidebar title updates end

5 · Document Ingestion — Sequence Detail

sequenceDiagram participant U as 👤 User participant FE as Frontend participant BE as FastAPI participant DC as Docling participant OL as Ollama · Gemma4 Vision participant EM as BGE-M3 + BM25 participant QD as Qdrant participant DB as draxie.db U->>FE: Drops file / clicks paperclip FE->>BE: POST /documents (multipart) BE-->>FE: {"status": "processing", "filename": "…"} BE->>BE: save to tmp file Note over BE: Background task starts (_ingest_file) BE->>BE: copy to data/uploads/ BE->>DC: DocumentConverter.convert(tmp_path) DC-->>BE: DoclingDocument BE->>DC: HybridChunker.chunk() + contextualize() DC-->>BE: raw chunks loop each chunk alt chunk is text/table BE->>BE: RecursiveCharacterTextSplitter else chunk is figure/chart BE->>BE: _extract_images() → PNG → data/images/ BE->>OL: _describe_image(image_path, surrounding_text) OL-->>BE: German description (Zahlen, Inhalt, Art) end end alt .docx file BE->>BE: _extract_docx_links() → hyperlink chunk end BE->>EM: embed all chunks (dense + sparse) EM-->>BE: vectors BE->>QD: vectorstore.add_documents(chunks) QD-->>BE: ok BE->>DB: INSERT ingestion_log (filename, chunks, status) FE->>BE: GET /documents/status (polling) BE-->>FE: {"status": "ok", "chunks": N} FE-->>U: Upload confirmed
